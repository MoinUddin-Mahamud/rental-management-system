# Rental Management System

## Project Overview

The Rental Management System is a comprehensive SQL Server database project developed to manage rental properties, tenants, payments, maintenance requests, and overall rental operations efficiently. The system demonstrates practical implementation of relational database concepts and advanced T-SQL programming techniques.

This project focuses on structured database design, transaction management, reporting, query optimization, and real-world rental management scenarios.

## Database Tables

| Table Name | Description |
|---|---|
| Users | Stores owner and tenant information |
| Properties | Contains rental property details |
| Rentals | Tracks rental agreements and tenant assignments |
| Payments | Stores payment and transaction records |
| MaintenanceReq | Handles maintenance requests and statuses |
| Images | Stores property-related images |
